# 项目安装
yarn install
# 项目运行
yarn run dev
# 项目打包
yarn run build

### 注意事项
本项目使用技术栈 vue3 + vite + pinia + element-plus组件库
1、node版本使用12以上
2、全局状态管理请使用pinia而不是vuex
3、前端时间格式化尽量使用dayjs来操作
